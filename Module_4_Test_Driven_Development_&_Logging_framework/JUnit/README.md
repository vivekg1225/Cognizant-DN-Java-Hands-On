# Module 4 - JUnit Hands-On

## Objective

This module demonstrates the fundamentals of unit testing in Java using **JUnit**. The exercises cover project setup, writing unit tests, using assertions, and organizing tests using the Arrange-Act-Assert (AAA) pattern.

---

## Technologies Used

- Java
- JUnit 4 / JUnit 5
- Maven
- IntelliJ IDEA / Eclipse

---

## Exercises

### Exercise 1: Setting Up JUnit

#### Scenario
You need to set up JUnit in your Java project to start writing unit tests.

#### Tasks
- Create a new Java project.
- Add the JUnit dependency to `pom.xml`.
- Create a new test class.

---

### Exercise 2: Writing Basic JUnit Tests

#### Scenario
Write unit tests for a simple Java class.

#### Tasks
- Create a Java class with methods to test.
- Write JUnit test cases for each method.

---

### Exercise 3: Assertions in JUnit

#### Scenario
Use different JUnit assertions to validate test results.

#### Assertions Used
- assertEquals()
- assertTrue()
- assertFalse()
- assertNull()
- assertNotNull()

---

### Exercise 4: Arrange-Act-Assert (AAA) Pattern

#### Scenario
Organize test cases using the AAA pattern and demonstrate setup and teardown methods.

#### Tasks
- Arrange test data.
- Perform the action.
- Assert the expected result.
- Use `@Before` and `@After` methods.
