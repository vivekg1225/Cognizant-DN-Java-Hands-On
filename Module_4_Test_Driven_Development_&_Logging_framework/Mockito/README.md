# Module 5 - Mockito Hands-On

## Objective

This module demonstrates the basics of **Mockito**, a popular Java mocking framework used for unit testing applications that depend on external services or objects.

---

## Technologies Used

- Java
- JUnit 5
- Mockito
- Maven

---

## Exercises

### Exercise 1: Mocking and Stubbing

#### Scenario

You need to test a service that depends on an external API. Use Mockito to mock the external API and stub its methods.

#### Tasks

- Create a mock object.
- Stub methods to return predefined values.
- Write a test case using the mock object.

---

### Exercise 2: Verifying Interactions

#### Scenario

Ensure that a method is called with the expected arguments.

#### Tasks

- Create a mock object.
- Invoke the service method.
- Verify that the required method is called.
