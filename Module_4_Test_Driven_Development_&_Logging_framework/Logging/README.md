# Module 6 - Logging with SLF4J

## Objective

This module demonstrates how to use **SLF4J** with **Logback** to implement logging in Java applications. Logging is an essential practice for debugging, monitoring, and maintaining software.

---

## Technologies Used

- Java
- Maven
- SLF4J
- Logback

---

## Exercise

### Exercise 1: Logging Error Messages and Warning Levels

#### Scenario

Write a Java application that demonstrates logging error messages and warning messages using SLF4J.

#### Tasks

- Configure SLF4J and Logback dependencies.
- Create a logger.
- Log warning messages.
- Log error messages.
